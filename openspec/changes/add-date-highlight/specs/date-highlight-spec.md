# Spec: 日期高亮功能

## Requirements

### R1: 颜色预设

- 提供 6 种预设高亮颜色，符合动森UI风格
- 颜色列表：薄荷绿、蜜桃粉、薰衣草紫、柠檬黄、珊瑚橙、天空蓝

### R2: 设置高亮

- 在日历页面长按或点击日期格子，弹出颜色选择面板
- 选择颜色后，该日期显示对应颜色的背景高亮
- 已有高亮的日期再次操作可更换颜色或移除高亮

### R3: 显示高亮

- 日历渲染时，检查每个日期是否有高亮数据
- 有高亮的日期显示半透明彩色背景
- 高亮不影响日期原有的 today/selected 样式逻辑

### R4: 数据持久化

- 高亮数据存储在 localStorage 的 calendarData 中
- 格式：`highlights: { "2026-06-22": "#5cc8b9", "2026-07-01": "#f0a0b0" }`

## Scenarios

### S1: 设置日期高亮

1. 用户在日历页面点击某个日期
2. 弹出颜色选择面板（含预设颜色圆点 + 移除按钮）
3. 用户选择一个颜色
4. 日期格子立即显示该颜色的半透明背景
5. 数据保存到 localStorage

### S2: 移除日期高亮

1. 用户点击已有高亮的日期
2. 弹出颜色选择面板，当前颜色高亮显示
3. 用户点击"移除"按钮
4. 日期恢复正常显示
5. 数据从 localStorage 中删除

### S3: 跨月保持

1. 用户设置某日期高亮
2. 切换到其他月份再切回来
3. 高亮颜色仍然正确显示
